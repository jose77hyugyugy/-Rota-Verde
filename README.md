# 🚛 Rota Verde

## Descrição do Projeto

O Rota Verde é um simulador educativo que permite analisar o impacto ambiental do transporte de cargas no campo. O usuário escolhe o tipo de veículo e a distância percorrida para calcular a emissão estimada de dióxido de carbono (CO₂).

O objetivo é conscientizar sobre a importância da logística sustentável e do uso de tecnologias que reduzam os impactos ambientais no agronegócio.

---

## Objetivo do Projeto

Demonstrar como a escolha dos meios de transporte pode influenciar diretamente a sustentabilidade das operações agrícolas, incentivando práticas mais responsáveis e eficientes.

## Como Executar

1. Faça o download ou clone este repositório.
2. Abra o arquivo `index.html`.
3. Escolha o transporte e a distância.
4. Clique em "Calcular Impacto Ambiental".
